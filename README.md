# PictureToWebP
图片转WebP格式，是对[webp-io](https://github.com/biezhi/webp-io)的封装

## 功能描述

支持单图片文件，或者批量图片文件转换成WebP格式文件

## 使用说明

下载此jar包（[传送门](https://github.com/LeacHar/PictureToWebP/releases/tag/V1.0)）

java -jar CoverWebP.jar
